# Source Map

## Purpose

Read this file when local project files do not fully answer a KubeJS question and you need authoritative documentation.

## When To Look Beyond The Core Wiki

- If the task mentions Create, Mekanism, LootJS, EMI, JEI, or other addon-specific behavior, inspect the addon list in the local modpack and then use the corresponding KubeJS addon page.
- If the project uses legacy syntax or older Minecraft versions, check version notes before suggesting a migration.
- If a wiki page is sparse, inspect the local project for working examples before inventing unsupported APIs.
